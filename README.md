# VCCGenerator

## Descripción

**VCCGenerator** es una aplicación web que permite generar y validar números de tarjetas de crédito y débito virtuales utilizando el algoritmo de Luhn.
Esta herramienta está destinada a fines educativos y de prueba, y su uso debe ser siempre ético y legal.

## Estructura del Proyecto

```

### Archivos y Carpetas

- **css/styles.css**: Contiene los estilos CSS para la aplicación.
- **html/index.html**: Página de inicio con información sobre la aplicación.
- **html/generator.html**: Página del generador de tarjetas.
- **html/validator.html**: Página del validador de tarjetas.
- **html/warning.html**: Página de advertencia sobre el uso ético de la herramienta.
- **img/Tarjeta.png**: Imagen utilizada en la página de inicio.
- **js/script.js**: Archivo JavaScript que contiene la lógica para generar y validar las tarjetas.

## Uso

### Generador de Tarjetas

1. Navega a la pestaña "Generador de Tarjetas".
2. Selecciona el tipo y categoría de tarjeta.
3. Ingresa el saldo deseado.
4. Haz clic en "Generar Tarjeta".
5. La tarjeta generada aparecerá en la sección de información generada.

### Validador de Tarjetas

1. Navega a la pestaña "Validador de Tarjetas".
2. Selecciona el tipo y categoría de tarjeta.
3. Ingresa el número de tarjeta, fecha de expiración, CVV y saldo actual.
4. Haz clic en "Validar Tarjeta".
5. El resultado de la validación aparecerá en la sección de resultados.
